Detailed Description

template<typename T>
class WorldMatrix< T >

2D rectangular vector in world space

vector up-left = {0,0} down-right = {width, height}

WorldMatrix down-left = {0,0} up-right = {width, height} transposed vector[4][5] to WorldMatrix

vector<T>{ {1,2,3,4} {1,2,} {1,2,3} {1,2,3,4,5} }

Versus

WorldMatrix<T>{ {1,1,1,1,0} {2,2,2,2,0} {3,0,3,3,0} {4,0,0,4,0} {0,0,0,0,5} }

This way the hardcoded vector has same orientation as the world space and potentially eases the mirroring and randomizing the elements WorldMatrix is not recommended to be used in tight loops left-right = Rows down-up = Columns

Axis Directions: WorldMatrix[Rows][Cols] (Row Major)

  • left-right = Rows
  • down-up = Columns

2D vector in world space Acts like a 2D vector but the code-space down-up axis is flipped then axes are transposed and padded
